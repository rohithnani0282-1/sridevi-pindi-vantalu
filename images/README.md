# Images Folder

This folder contains images for the SRIDEVI PINDI VANTALU website.

## Required Images:

### pindi-vantalu-background.jpg
- **Purpose**: Hero section background image
- **Location**: Used in the main hero section
- **Recommended Size**: 1920x1080 pixels or higher
- **Format**: JPG format for optimal loading
- **Content**: Traditional Pindi Vantalu items, pickles, sweets, or South Indian food spread

## Instructions:

1. **Add your image**: Place a high-quality Pindi Vantalu photo in this folder
2. **Name it correctly**: Rename the file to `pindi-vantalu-background.jpg`
3. **Optimize**: Ensure the image is web-optimized (under 2MB if possible)
4. **Test**: Refresh the website to see the new background

## Image Guidelines:

- Use bright, appetizing food photography
- Include variety of pickles, sweets, and snacks
- Ensure good lighting and composition
- Avoid watermarks if possible
- Landscape orientation works best

## Alternative:

If you don't have an image ready, the website will currently show a placeholder gradient background until you add the actual image file.
